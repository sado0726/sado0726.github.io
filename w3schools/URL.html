URL = Uniform Resource Locator Internet Protocal (IP) address http://www.w3schools.com/html/default.asp scheme://host.domain:port/path/filename
scheme - defines the type of Internet service (most common http)

http common web pages. not encrypted
https secure web pages. encrypted
ftp downloading or uploading files
file a file on your computer

host - defines the domain host (default host for http is www) domain - defines the Internet domain name (w3schools.com) port - defines the port number at the host (default for http is 80) path - defines a path at the server (if omitted: the root directory of the site) filename - defines the name of a document or resource 

nit picky stuff about URL encoding:
	URLs can only be ASCII
	URL encoding replaces non ASCII character with a "%" followed by hex
	URLs cannot contain space so it replaces all space with plus (+) sign or %20

XHTML stands for Extensible HyperText Markup Language
almost identical to HTML but stricter and is supported by all major browsers
By combining HTML and XML together, XHTML was developed.
Few mandatory concepts:
DOCTYPE necessary, html, head, title, body necessary, html must have something in it.
properly nested, closed, lowercase, one root elemtn, attribute names lowercase and quoted, and can't be minimized
